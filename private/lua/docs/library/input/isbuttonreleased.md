# input.IsButtonReleased
Checks if an button is released by keycode.

## Syntax
```
input.IsButtonReleased(keycode:int)
```

## Parameters
```keycode``` Keycode to check for. 

## Example
```lua
if input.IsButtonReleased(1) then
	print("Left click is released!");
end
```
