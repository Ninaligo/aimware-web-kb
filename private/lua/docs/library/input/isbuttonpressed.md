# input.IsButtonPressed
Checks if an button is pressed by keycode

## Syntax
```
input.IsButtonPressed(keycode:int)
```

## Parameters
```keycode``` Keycode to check for. 

## Example
```lua
if input.IsButtonPressed(2) then
	print("Right click is pressed!");
end
```
